# Contributing to LockFS

## Bug Reports

- Search for existing issues
- If the topic does not exist, open a new [issue](https://github.com/ghost-in-a-jar-00/LockFS/issues)
- Write in the following format
  `- Title: Summary of issue`
  `- Description: Elaborate on the issue in detail(i.e. how to recreate, which platform you are using)`

## Feature Requests

**If you have a suggestion for a feature to be implemented on LockFS, please discuss with us with the following guidelines:**

- Search for existing issues
- If the topic does not exist, open a new [issue](https://github.com/ghost-in-a-jar-00/LockFS/issues) and explain with the following
  - `Title: Requested feature`
  - `Description: Explain the motivations of the feature, how it works`
- Fork and submit a pull request if you are able to implement the feature

## Pull Requests

**Here are some guidelines to follow when submitting a pull request:**

- Use a prefix to summarize your pull request in the title
  - E.g. `*Bug Fix: <bug to be fixed>*`
- Explain what you have improved on, added and why
- Wait for review
